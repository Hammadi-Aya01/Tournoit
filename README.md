# 🏆 Tournoit - e-Sport Tournament Platform

A Java web application for managing e-sport tournaments.  
The project provides services for tournament organization using REST/SOAP APIs, database persistence, and a web interface.

---

## ✨ Features

- Tournament management
- Team and player management
- Match organization
- REST API services
- SOAP web services
- Database persistence with Hibernate
- H2 database integration
- Web interface

---

## 🛠️ Technologies

- Java 17
- Maven
- Jakarta EE
- Jersey REST
- JAX-WS SOAP
- Hibernate ORM
- H2 Database
- JavaFX
- HTML / CSS / JSP

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/Hammadi-Aya01/Tournoit.git
cd Tournoit/demo
```

### Build the project

```bash
mvn clean install
```

### Run / Deploy

The project is packaged as a WAR file and can be deployed on a Jakarta EE compatible server.

```bash
mvn clean package
```
